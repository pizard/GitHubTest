 	 - 세분화된 ID 및 액세스 관리

 	 - ID를 사용하여 액세스 보호

 	 - 민감한 데이터 탐색 및 수정

 - Cloud Resource Manager
 	 - GCP에서 계층적으로 리소스 관리
 - Cloud Security Scanner
 	 - App Engine 앱 자동 스캔
