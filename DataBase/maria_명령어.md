# mysql 명령어
## 오늘 사용한 명령어(나중에 다시 정리)
 - cmd창에서 mysql 접속
 	 - `mysql -h 호스트명 -u 아이디 -p`
 	 - 이후 암호 입력
 - 프로시져 목록 확인
 	 - `show procedure status `
 - 프로시져 스크리븥 확인
	 - `show create procedure uso_Get_Mini_Evaluation`
